# HeartlandStormChaser

Cloudflare-ready storm chase operations foundation — Workers, static assets, and D1.

**Phase 1** (in progress): project scaffold, static pages, health API, D1 schema, workflow docs.

## Quick start

```bash
npm install
npx wrangler --version
```

Full local development, database, Git, and deployment steps will be documented in [`docs/WORKFLOW.md`](docs/WORKFLOW.md) as each phase is built.

## Stack

- Cloudflare Workers (API)
- Cloudflare static assets (HTML/CSS/JS)
- Cloudflare D1 (SQLite)
- Wrangler CLI
- GitHub
